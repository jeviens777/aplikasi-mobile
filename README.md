# 📍 Profile Card — Smart Check-In App

Aplikasi mobile (React Native/Expo) untuk check-in dengan foto kamera + deteksi lokasi GPS otomatis.

## ✨ Fitur Utama
- 📸 Smart Check-In — ambil foto kamera + GPS otomatis dalam satu tombol
- 🗺️ Deteksi alamat otomatis dari koordinat GPS (reverse geocoding)
- 🖼️ Pilih foto dari galeri
- 🔒 Permission handling kamera & lokasi
- ♻️ Reset data
- 🏷️ Menampilkan versi app di UI
